# XRGL

XRGL is a research software project for integrating augmented reality visualizations into data science notebooks using Microsoft HoloLens 2.

This repository accompanies the thesis:

**Derek Willis. _Integrating augmented reality visualizations into data science notebooks using Microsoft HoloLens 2._ Mississippi State University, 2026.**

## Overview

XRGL supports a notebook-native workflow for creating 3D visualizations in a familiar data science environment, exporting them for augmented reality viewing, and inspecting them on Microsoft HoloLens 2.

The workflow is designed around a simple pipeline:

**construct → extract → encode → transfer → view**

Rather than replacing the notebook environment, XRGL treats augmented reality as an extension of the existing visualization process. Users can continue working in a traditional 2D notebook while using AR for immersive spatial inspection when beneficial.

## Features

- Creates 3D scenes within a notebook-based workflow
- Supports export of visualization content for AR viewing
- Enables inspection of 3D content on Microsoft HoloLens 2
- Preserves a lightweight workflow centered on reproducibility and extension
- Supports research and experimentation in immersive analytics and notebook-based visualization

## Research Context

XRGL was developed as part of graduate research exploring how augmented reality can be integrated into data science notebook workflows without requiring users to abandon conventional desktop tools and interfaces.

The goal is to support a practical mixed workflow in which:
- 2D notebooks remain the primary analytic environment
- 3D visualizations can be exported when immersive inspection is useful
- augmented reality enhances, rather than replaces, existing analysis practices
